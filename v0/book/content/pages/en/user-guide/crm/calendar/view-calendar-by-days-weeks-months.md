## What each view is for

* **Day** – Focus on one specific day and its scheduled activities.
* **Week** – Get a practical overview of activities across the current week.
* **Month** – See long-term planning and event distribution in a monthly grid.
* **List** – Display events as rows sorted by date for quick scanning.

## Tips

Use **Week** view for regular operational planning.

Use **Month** view to spot busy periods and scheduling gaps.

Use **List** view when you need a compact chronological overview.

## Summary

| View mode | Best use |
| --------- | -------- |
| Day | Detailed planning for a single day. |
| Week | Team coordination and short-term planning. |
| Month | High-level planning and workload distribution. |
| List | Fast date-based overview of all events. |
