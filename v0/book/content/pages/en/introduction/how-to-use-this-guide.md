# How to use this guide

This guide is designed to be your primary resource for learning, navigating, troubleshooting, and mastering Hubleto. Whether you are using Hubleto for the first time or already work with the platform regularly, the sections and visual conventions described below will help you find the right information faster.

## Finding information

The documentation is organized so that you can access information in several ways, depending on what you are looking for.

**Navigation tree**

Use the sidebar on the left side of the page to browse topics by category. The navigation tree groups related pages together, so you can move from general topics to more specific app articles.

**On this page**

Longer articles include an **On this page** table of contents on the right side of the page. Use it to jump directly to a specific section without scrolling through the whole article.

**Related topics**

Many pages include a **See also** section near the bottom. This section links to related articles that can help you continue learning about the same feature or workflow.

## Visual cues and formatting

This guide uses consistent formatting to make instructions easier to read and follow.

| Element | Meaning |
| ------- | ------- |
| **Bold text** | Highlights important terms, buttons, menu items, or key information. |
| `Monospace text` | Represents file paths, URLs, technical values, or commands that should be typed exactly. |
| `Key + Key` | Represents keyboard shortcuts that help you perform actions faster. |

## Information blocks and guide sections

Some pages contain repeated sections that provide additional context or practical instructions.

**Examples** provide extra context and show how a feature may be used in a real situation.

**How to** sections explain practical workflows step by step, usually for common tasks that users perform in the application.

**See also** sections point to related pages so you can explore connected topics without searching manually.

## Getting started

If this is your first time using Hubleto, start with the **Getting Started** section in the left sidebar. It covers the essential first steps, basic navigation, and initial setup topics that help you begin using the platform quickly.

After that, continue with the **Apps** section to learn how to work with individual modules such as CRM, Sales, Projects, Finance, and Maintenance.
