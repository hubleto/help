# What is Hubleto

Hubleto is an open-source **Business Application Hub** designed to bring multiple business processes into one unified platform. It combines ready-to-use business applications with a flexible development framework, so companies can manage daily operations and extend the system when their requirements grow.

Instead of being only a fixed software product, Hubleto is built as a modular application suite. It can support common CRM (Customer Relationship Management) and ERP (Enterprise Resource Planning) workflows, such as working with customers, contacts, calendars, deals, orders, projects, documents, reports, and settings.

At its core, Hubleto is built to grow with your business. Whether you need a simple tool to manage contacts or a complex system to oversee global production and billing, Hubleto provides the foundation to build, customize, and scale your digital infrastructure.

## Key characteristics

| Characteristic | What it means |
| -------------- | ------------- |
| Open source | Hubleto is built on open-source principles, which means the source code can be inspected, adapted, and extended. |
| Modular architecture | The platform is composed of applications, so you can use the modules you need and extend the system with additional apps. |
| Flexible hosting | Hubleto can be deployed in a cloud environment or installed on your own infrastructure, depending on your security and operational requirements. |
| Responsive interface | The user interface is designed to work across desktop and mobile devices, so the platform can be accessed from different working environments. |

## Core capabilities

Hubleto helps reduce data fragmentation by keeping related business information in one place.

**Business process management**

Teams can use Hubleto to manage operational workflows, records, tasks, and activities. This helps keep work organized and makes it easier to follow what needs to be done next.

**CRM and ERP functionality**

Hubleto includes applications for customer management, contacts, calendar activities, sales-related records, projects, documents, reports, and other business areas. These applications are designed to work together instead of storing information in separate isolated tools.

**Reporting and dashboards**

Dashboards and reports help teams understand performance, review business activity, and make better decisions based on available data.

**Extensibility for developers**

Hubleto is also a development framework. Developers can create custom applications and adapt existing modules to match specific company processes.

## Why use Hubleto?

Many business systems require companies to adapt their processes to the software. Hubleto takes a more flexible approach: the platform can be extended and adjusted to better match how your company actually works.

Because of its modular structure, you can start with the features you need today and add more functionality later. This makes Hubleto suitable both for smaller teams that need simple business tools and for organizations that require more customized workflows.
